📚 高三知识点掌握情况调研

高二升高三 · 六科全维度学情分析本地模式

🎓

全面把脉 · 精准提升

覆盖语文、数学、英语、物理、化学、生物六大科目，共计60个核心知识模块，帮助学生、家长、老师全面了解学习状况，有针对性地进行指导。

📋 调研说明

  • 语文（写作强化）：11个知识模块，覆盖阅读基础、语用基础、议论文写作、其他写作、写作提升
  • 数学：10个知识模块，覆盖函数与导数、三角与解三角形
  • 英语（写作强化）：10个知识模块，覆盖语法基础、阅读题型、应用文写作、读后续写
  • 物理：10个知识模块，覆盖力学基础、能量与动量、力学综合、电磁学开篇
  • 化学：10个知识模块，覆盖基础原理、电化学、元素化合物、反应原理
  • 生物（遗传核心）：10个知识模块，覆盖基础铺垫、遗传定律、分子遗传、变异与育种、遗传综合
  • 四个量级：完全掌握 / 基本掌握 / 部分掌握 / 不会
  • 三种视角：老师视角（班级全局）、家长视角（个体对比）、学生视角（自我提升）
  • 一键分析：支持提交AI进行深度分析，点击即可获取专业教学建议

💡 使用流程

  • 每位学生/家长点击「开始填写调研」，输入姓名后逐科评估
  • 填写完成后数据自动保存在本设备，可随时查看统计
  • 如需汇总多人数据：用「导出数据」生成分享码，发送给汇总人「导入数据」
  • 在统计页面选择老师/家长/学生视角，点击「提交AI分析」获取AI建议

🔧 首次使用 — 配置 Supabase 云数据库

当前为本地存储模式，数据仅保存在当前浏览器。配置 Supabase 后可实现多设备实时共享。

  1. 访问 supabase.com 注册免费账号
  2. 创建新项目（选择 Free 计划）
  3. 进入 SQL Editor，运行 supabase-setup.sql 脚本
  4. 进入 Settings → API，复制 Project URL 和 anon key
  5. 填入下方，点击「保存配置」
